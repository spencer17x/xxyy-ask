# Feature Status

本文档记录 XXYY 客服应用当前已完成和计划中的用户、客服与运营可感知功能。

## User Features

用户可感知功能：

- [x] 产品功能问答：回答 XXYY 产品功能、配置步骤、Pro 权益、文档内容和官方 X 更新相关问题。
- [x] 来源引用展示：每条知识库回答都会返回引用来源，避免无依据编造产品细节。
- [x] 流式回答：Web UI 支持边生成边展示回答。
- [x] 视频附件展示：知识库片段里包含视频链接时，Web UI 可以展示视频附件。
- [x] 回答反馈：用户可以对回答提交正负反馈。
- [x] 边界问题回复：账户余额、订单、交易记录、MEV/夹子检测、投资建议等问题目前会返回边界说明，不会假装查询实时数据。
- [ ] 多轮对话：支持基于当前会话继续追问、引用上下文和保留用户偏好。
- [ ] 交易哈希夹子检测：用户直接发送交易哈希后，系统判断是否被夹，并返回被夹相关信息和关键交易路径。
- [ ] 夹子检测截图返回：在夹子检测结果里展示或返回交易分析截图。
- [ ] 工单创建：无法直接解决的问题可以创建客服工单。
- [ ] 人工客服接管：复杂问题可以转人工继续处理。
- [ ] 多渠道接入：支持 Telegram、Discord、站内 widget、移动端等入口。

## Admin / Ops Features

客服和运营可感知功能：

- [x] 运维页面：提供 `/ops` 同源运维页。
- [x] 服务健康检查：提供 `/health` 和 `/health/deep`，用于检查服务、数据库、embedding 和 LLM 配置状态。
- [x] 知识库统计：支持查看文档数、chunk 数、来源 URL 数、最新更新时间和最近 ingestion run。
- [x] 反馈统计：支持查看正负反馈数量和最近反馈明细。
- [x] 负反馈导出：支持导出负反馈队列，用于补充知识库或扩展评测集。
- [x] 知识库同步：支持产品文档 ingest、官方 X 更新增量同步和全量重建。
- [x] RAG 评测：内置产品客服回归评测集，可用于发布前检查回答质量。
- [ ] 工单后台：查看、分配、处理和关闭客服工单。
- [ ] 会话后台：查看用户会话、检索来源、反馈和处理结果。
- [ ] 知识库管理后台：支持文档编辑、审核、发布和知识缺口归类。
