# Avg. Price Line

## Provenance

This is a reviewed English translation derived from the [official Chinese documentation](https://docs.xxyy.io/getting-started/k-xian-qu-yu/ping-jun-mai-ru-cheng-ben-xian) because the corresponding English page currently contains only a heading.

## Product behavior

XXYY calculates an average cost line from every purchase. When a token is bought multiple times, XXYY recalculates the average purchase cost so the user can compare the current price with the cost basis of the position.
