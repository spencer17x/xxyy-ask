# 收益统计

收益统计会筛选当前交易地址的交易信息，将所有交易记录详细展示出来，包括交易时间、类型、价格、购买总额以及Sol总额，并且计算当前交易地址的持仓情况以及利润PnL等

与此类似的是，右侧持仓面板会显示总买入术量、, 数量、持有中的所数量以及总卖出数量和 Pnl 收益统计里的信息会更加完善一些，展示了更加详细的交易记录以及代币价格的实时变化。
