# 批量导入设置

<figure><img src="/assets/xxyy-docs-M2KSZxlxcOG2NPpU9zSN.png" alt=""><figcaption></figcaption></figure>

批量导入支持文字导入或文件导入两种类型。

文字导入需按照 钱包名称,钱包地址,分组的格式进行设置，如有错误信息会提示导入失败或错误。

文件导入网站上有提供导入钱包模板，下载导入钱包模板或直接上传 Abot 钱包文件可直接进行导入

导入时可批量设置该批钱包的设置类型，是否只查看买入或卖出，是否开启推送
