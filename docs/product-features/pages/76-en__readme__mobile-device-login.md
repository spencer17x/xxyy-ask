# Mobile Device Login

Once you are logged into the XXYY website, you can easily connect your mobile device.

1\. Access the Connection Page\
In the top navigation bar on the website, find and click the "Mobile App" button. This will take you to the mobile authorization page, where a QR code will be displayed.

2\. Scan and Verify\
Use your mobile device to scan the QR code on the authorization page. This will open a verification page on your mobile device.

* The QR code on the website is valid for 60 seconds and will refresh automatically.
* A unique verification code will appear on your mobile screen after you scan the QR code.

3\. Complete the Login\
Enter the verification code displayed on your mobile device into the input field. This will automatically log you into XXYY on your mobile device using the same wallet that is connected to your web session.

If the verification code expires before you can use it, you must re-scan the new QR code on the website to receive a new one.

Important Security Feature: Synchronized Sessions

To ensure the security of your trades and assets, your login status is synchronized across all devices.

* For your protection, logging out from one device (either mobile or web) will automatically log you out from all other active sessions simultaneously.
