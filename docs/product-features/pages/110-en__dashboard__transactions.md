# Transactions

The Transactions feed displays a real-time log of every trade in the token's pool, with a delay of less than one second from the blockchain.

The log includes the following columns: Date, Type, Price (SOL), Total (USD), Total (SOL), Makers, and Action. The Action column for each trade allows you to:

* Filter all transactions from that specific wallet address.
* View the transaction hash on a block explorer like Solscan.
* Monitor the address by adding it to your Monitor list.

Use the filters at the top of the panel to sort the list by time, filter by transaction type (e.g., Buys only), toggle between SOL and USD values, filter by trade size, or search for a specific wallet address.
