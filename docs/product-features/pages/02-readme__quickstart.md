# 新手必看

XXYY支持Solana链的代币交易。如果你是该领域的新手，建议你在开始交易前先了解以下内容。

* 登录钱包：您用于连接登录XXYY网站的钱包，该钱包仅用于授权登录。
* 交易钱包：您在XXYY网站上生成的钱包，主要用来交易代币。
* 滑点：滑点的大小决定您在每笔交易中实际成交价格的波动范围，滑点越高能够接受的价格波动越大。
* 交易Fee：能够影响交易上链的速度，更高的交易Fee可以在区块中获得更早的位置。
