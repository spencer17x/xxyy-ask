# Holder

<figure><img src="/assets/xxyy-docs-rLFlxJvymmD0CnN5zQ3o.png" alt=""><figcaption></figcaption></figure>

Holder页面会展示当前代币持有者的所有地址汇总情况，支持查看持仓量前100的所有地址。

* Rank

Rank是指当前地址持仓量在该项目的实时排名

* 钱包地址

钱包地址支持展示，关注备注名高亮显示以及展示其买入卖出次数。后面会标记该钱包的行为，具体的标记展示信息参考下面的表格。方便通过标记快速识别阴谋盘和庄的真实持仓。

* 持仓量

持仓量是指地址在这个项目的持有代币数量

* 持仓价值

持仓价值是指该持有代币的价值，单位是 USD

* 占比（真实持仓）

持仓占比是指该地址持仓代币数量占总供应量的占比。真实持仓是指有转账或分仓行为。XXYY 会统计出其转账分仓的真实持仓占比，并以黄色标记高亮显示

* SOL来源

Sol 来源是指该地址第一笔SOL 的转账的交易地址，有可能是CEX交易所或某个SOL 地址

* Token来源

Token来源会显示是购买或转账，如是转账则会显示转账地址

* 余额（SOL）

余额会显示当前地址所有的SOL 余额，来判断该地址是否为巨鲸钱包或值得关注的钱包

* 最后交易时间

最后交易时间是指该地址在该代币上最后操作的时间。
