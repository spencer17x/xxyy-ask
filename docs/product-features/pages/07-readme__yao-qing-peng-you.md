# 邀请朋友

* 邀请规则：当注册进入网站后，将会自动生成的一条带有链信息和唯一参数的邀请链接，邀请参数仅可修改一次，修改后无法再次修改。邀请朋友注册后，对方在 XXYY 上交易手续费的10%将作为你的邀请奖励。
* 邀请和奖励记录：支持查看累计邀请奖励，以及提现记录。
* 邀请返佣提现：已经获得的奖励金额可以申请提现，提现奖励将在次日UTC 0 点 转入选择的交易钱包。
* 如果之前使用过 XXYY，无法再次绑定邀请，或者换绑其他帐号。
