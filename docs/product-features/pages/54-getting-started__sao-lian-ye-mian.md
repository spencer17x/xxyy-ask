# 扫链页面

<figure><img src="/assets/xxyy-docs-BJY6y55V0JvaIYolgP1o.png" alt=""><figcaption></figcaption></figure>

### ⚙️ 交易设置

可选择交易钱包，快捷设置4个买入金额，选择交易模式、交易费和滑点

### 🆕 新交易对

新交易对是指Pump项目新发射的所有项目

### ⏰ 即将打满

即将打满是指Pump项目根据进度倒序排列的项目

### 🚀 已经发射

已经发射是指Pump上raydium的项目
