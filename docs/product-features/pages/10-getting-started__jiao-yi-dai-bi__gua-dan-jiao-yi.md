# 挂单交易

XXYY 提供挂单买卖功能，一键设置，按理想价格买入或卖出代币。

#### ⚙️ 挂单设置：

* 价格上涨：当前实时价格上涨百分比，设置买入或卖出
* 价格下跌：当前实时价格下跌百分比，设置买入或卖出
* 市值上涨：当前实时市值上涨百分比，设置买入或卖出
* 市值下跌：当前实时市值下跌百分比，设置买入或卖出
* 价格到达：价格到达设置买入或卖出，设置买入或卖出
* 有效时间：设置该挂单的有效时间（单位：小时）
