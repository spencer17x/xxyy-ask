# K 线交易标记

<figure><img src="/assets/xxyy-docs-WK0lNaxaDTjKOW6HFVtP.png" alt=""><figcaption></figcaption></figure>

K 线标记分为查看自己的交易和查看 Dev 标记两种，支持配合最新成交或 Holder 查看置顶地址的买入卖出标记。

Buy：买入标记

Sell：卖出标记

Dev Buy：Dev 买入标记

Dev Sell： Dev 卖出标记

Raydium：Pump 项目发射到 Raydium 的节点标记
