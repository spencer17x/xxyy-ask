# 持仓盈亏

若你持有某个代币，则在该代币的详情中将会展示持仓信息，包括买入价值，持仓中价值，卖出价值，收益。

* 买入价值（Invested）：买入代币的USD价值，Token数量/买入交易笔数
* 卖出价值（Sold）：卖出代币的USD价值，Token数量/买入交易笔数
* 剩余价值（Remaining）：剩余持有Token的USD价值
* 盈亏倍率（Change in P\&L）：（持仓价值+卖出价值-买入价值）/买入价值
* 分享：你可以选择向X，Telegram好友或者群分享你的持仓收益，也可以下载持仓收益图保存。若有朋友通过你分享的链接注册 XXYY 并进行交易，你将获得分佣奖励。
