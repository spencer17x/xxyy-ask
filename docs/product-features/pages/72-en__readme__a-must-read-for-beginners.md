# A Must-Read for Beginners

XXYY supports token trading on the Solana, Ethereum, and BNB Smart Chain (BSC). If you're new to this space, we recommend understanding the following concepts before you start trading.

* Login Wallet: This is the wallet you use to connect to the XXYY website. It is used only for login authorization.
* Trading Wallet: This is a new wallet generated for you on the XXYY website, used specifically for executing trades.
* Slippage Tolerance: This setting determines the maximum acceptable price change between when you submit a transaction and when it is confirmed on the blockchain. A higher slippage tolerance means you accept a wider range of price fluctuation.
* Transaction Fee (or Priority Fee): This fee influences how quickly your transaction is processed on the network. A higher fee can help your transaction get included in a block faster, especially during times of high network congestion.
