# 自动交易

&#x20;自动交易包含：

* 跟随Dev卖100%
* 开盘狙击
* Raydium自动卖

需要注意的是：自动交易的所有交易模式均与设置时下方勾选的模式有关，请谨慎选择交易模式。

推荐的交易模式和滑点为：防夹模式、2000% 滑点
