# Raydium自动卖

如果在 Pump 阶段买入对应代币，在代币添加 Raydium 流动性之前，可以创建自动卖出订单，自定义卖出比例。

创建成功后，当识别到代币在 Raydium 添加流动性池子，会第一时间发起卖出。
