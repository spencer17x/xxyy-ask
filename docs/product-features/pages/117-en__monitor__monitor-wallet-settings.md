# Monitor Wallet Settings

When you add a new wallet to monitor, you can configure the following settings to customize how you track it and what alerts you receive.

* Wallet Address: Enter the public address of the wallet you wish to track.
* Wallet Name: Assign a custom name or label (e.g., "Smart Money Whale" or the KOL's name) for easy identification.
* Set Wallet Group: Organize your monitored wallets by assigning them to a group. You can select an existing group or type a new name to create one.

Transaction Alert Settings

* Customize Txn Alerts: Select the specific types of transactions you want to be notified about. Options include Buy, Sell, Send SOL, Received SOL, Send Token, and Received Token.
* Min Buy Amount: Set a minimum value (in SOL) for buy transactions. You will only receive an alert if a purchase exceeds this amount. This helps reduce noise from small trades.
* Min Sell Amount: Similarly, set a minimum value for sell transactions to only be alerted for significant sells.
* Enable Txn Alerts: Use this master toggle to turn all notifications for this specific wallet on or off. You can follow a wallet's activity in the feed without receiving constant push alerts.
* Enable alerts only for Pump: When toggled on, you will only receive alerts for this wallet's transactions that occur on the pump.fun protocol.
