# Snipe

Search for a token's contract address. You can set up a snipe order under the following conditions:

1. The token has not yet added liquidity to a Raydium pool.
2. The token is a pump.fun project that has completed its bonding curve and is pending its liquidity migration to Raydium.

Once the snipe order is set, our system will monitor the blockchain. The instant the token's liquidity pool is created on Raydium, your buy order will be executed automatically, making you one of the first buyers.

To create a snipe task, simply input the amount you want to buy and click "Confirm Snipe".
