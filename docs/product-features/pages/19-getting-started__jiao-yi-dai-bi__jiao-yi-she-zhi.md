# 交易设置

在右上角菜单中点击设置后，进入交易设置页面，你可以设置以下参数，设置完成后，对应参数将立即生效。

* 滑点（Slippage）：支持设置自定义滑点比例，设置完成后交易组件中默认使用该值
* 交易模式（Trade Mode）：支持设置默认交易模式是否防夹，设置完成后交易组件中默认使用该值
* 交易 Fee（Priority Fee）：单位SOL，包含 Gas 费和 Tip 费，支持设置自定义优先费金额，设置完成后交易组件中默认使用该值
