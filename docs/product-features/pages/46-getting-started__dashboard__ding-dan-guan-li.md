# 订单管理

* 挂单
* 狙击
* Raydium自动卖
* Dev 跟 Sell

订单管理会展示所有待执行的订单操作及历史订单记录。

可根据订单状态判断成功失败执行中或已过期

支持点击订单最前面的删除按钮删除或取消订单。
