# Feature Updates

XXYY v1.4.5:

1. Twitter Monitoring: You can now access the new Twitter Monitor in the Discovery sidebar. Subscribe to specific Twitter accounts to receive instant alerts.
2. Tag Holder & KOL Filters: New filters for Tag Holders and KOLs have been added to the main discovery scan, improving the efficiency of identifying high-potential tokens.
3. Bags Claim Feed: A new "Bags Claim" list has been added to the Discovery section, showing the latest tokens claimed on the Bags protocol. (Note: Claim data may have discrepancies; verify with multiple sources).
4. Hover-to-View P\&L in Monitor: In the Monitor list, you can now hover your mouse over a wallet address to quickly see its P\&L for the corresponding project.
5. Enhanced Wallet Import: The Monitor's bulk wallet import is now compatible with address formats from GMGN, Axiom, Photon, Bullx, and more. Simply paste a list of addresses for automatic recognition.

XXYY v1.4.4:

1. USDT Pair Trading on ETH: The Ethereum chain now supports data querying and trading for tokens paired with USDT.
2. Wallet Address Search on SOL: You can now search directly for a Solana wallet address to quickly view its performance data.
3. Monitor List UI Upgrades: The Monitor list now displays token avatars (hover to enlarge) and includes a hover-to-pause alerts function.
4. Wallet-Specific Holdings View: The main Holdings page now allows you to filter the view by a specific wallet, making it easier to manage positions on mobile.
5. Referral History: The "Invite Friends" page now includes a history of your referrals and the commission rewards they have generated.
6. Browser Tab Display: The browser tab title can now be set to display either the token's Price or its Market Cap, based on your preference.

XXYY v1.4.3:

1. Bonk Protocol Speed Boost: Optimized performance for tokens on the Bonk launchpad, resulting in faster chart updates, transaction history, and monitoring.
2. "Early Buyer" Tag: A new "Early Buyer" tag has been added to the Holders list to identify the wallets that purchased the token first.
3. Real-Time Tags in Trades: The "Latest Transactions" feed now applies "Early Buyer" and "Dev" tags in real-time to help you track their activity.
4. Similar Tokens on Hover: In the token details view, you can now hover over a token's avatar to see a list of similar or related tokens.
5. Faster Filtered Results: Optimized the data refresh speed for filtered discovery lists, ensuring tokens that no longer match the criteria are removed in real-time.
6. Improved List Navigation: In the Discovery sidebar, you can now left-click to open a project in the current tab or right-click to open it in a new tab.

XXYY v1.4.2:

1. Search in Monitor Management: You can now search for a wallet by its name or address within the Monitor management page.
2. Sort by Balance in Monitor Management: The Monitor management list can now be sorted by a wallet's total SOL balance.
3. Axiom & Bullx Import Compatibility: The bulk wallet import feature now supports address formats from Axiom and Bullx.
4. Per-Chain Audio Alerts: Audio alerts for the Monitor list can now be enabled or disabled on a per-chain basis.
5. Faster Social Previews: Significantly improved the loading speed of Twitter profile and community previews.

XXYY v1.4.1:

1. Platform-Wide Speed Optimization: Continued improvements to page loading, transaction speed, and data refresh rates for a noticeably smoother experience.
2. New Wallet Data Page: You can now click on any wallet address in the "Latest Transactions" or "Holders" lists to view its detailed trading history.
3. Quick Buy in Discovery Feed: The multi-column discovery feed now supports quick-buy buttons for instant trading.
4. Synced Discovery Feed Data: The data in the multi-column discovery feed is now perfectly synchronized with the main scan page.
5. Real-Time Trade Counts & Tags: The "Latest Transactions" list now displays a wallet's total buy/sell count and tags in real-time.
6. Enhanced Twitter Previews: Improved the speed and reliability of Twitter profile and community previews.

XXYY v1.4.0:

1. Transaction Speed Optimization: Deep optimization of transaction speeds, especially for post-launch BONK trading, significantly improving success rates.
2. Market Cap Toggle in Transactions: The "Latest Transactions" feed can now be toggled to display Market Cap instead of price.
3. Meteora Tax Display: Buy and sell taxes are now displayed for tokens in Meteora DBC, DAMM V2, and DLMM pools.
4. Direct Editing in Tag Holder List: You can now edit a wallet's information directly from the Tag Holder list.
5. Faster Tweet Previews: Optimized the loading speed of tweet previews in the discovery scan list.
6. Option to Disable Price Risk Warning: You can now disable the price impact warning pop-up in the trade settings. Please use with caution.

XXYY v1.3.9: XXYY Scan 2.0 Launch!

1. All-New Scan Page UI: A completely redesigned UI for a better, faster, and more comprehensive scanning experience.
2. BSC & SOL Sync: The new experience has been rolled out simultaneously for both BSC and Solana.
3. Customizable Multi-Column Layout: Display custom discovery lists (Watchlist, Trending) alongside the three main scan columns.
4. Full Launchpad Coverage: Now supports all major launch protocols, including pump.fun, LetsBonk, Jupiter LFG, and more.
5. Exclusive KOL & Tag Holder Data: Leverage XXYY's unique data to track smart money.
6. Social Media & Ad Filters: Filter for tokens that have social media links or have paid for ads on Dexscreener.
7. Advanced On-Chain Filters: Utilize a new suite of filters covering key trading and security metrics.
8. Keyword Search: Search for tokens by name or use the filter to include/exclude specific keywords.
9. Block Token: You can now hide or "block" specific tokens from your scan lists.
10. Enlarged Avatars & Similar Tokens: Hover over an avatar to view a larger image, search Google, or find similar tokens.
11. Customizable Buy Buttons: Switch between a single or multi-button layout for quick buys.
12. Wallet Selection: Choose between using an XXYY trading wallet or your own self-custody wallet for trades.
13. Flexible Navigation: Left-click to open a token's details within the scan page, or right-click to open in a new tab.
14. Launch Audio Alert: Enable an audible alert for when a token in the "Final Stretch" list is about to launch.
15. Mobile-Friendly Design: The new scan page is fully optimized for mobile devices.
