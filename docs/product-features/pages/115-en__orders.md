# Orders

The Orders tab is your central hub for managing all of your automated and pending transactions. Here, you can track both active orders and view your complete order history.

You can filter the view by the specific type of order using the sub-tabs:

* Limit: Standard limit buy/sell orders.
* Snipe: Automated buys for new token launches.
* Graduated Sell: Automated sells for tokens migrating from a launchpad.
* Dev Sell: Automated sells that trigger on Dev activity.

Managing Your Orders

The Status column shows the current state of each order, such as Active, Completed, Failed, or Expired.

To cancel a pending order, simply click the delete icon on the left-hand side of its row.

Understanding the Order Details

The list provides a detailed breakdown of each order:

* Token: The token the order is for.
* Amount: The quantity of the token to be bought or sold (e.g., 100%).
* Condition: The specific trigger for the order (e.g., Take profit at ≥ 20%).
* Created When: The price of the token when the order was created.
* Target Price: The price at which the order is set to execute.
* Status: The current state of the order.
* Slippage: The slippage tolerance set for this specific order.
* Trading Fee: The transaction fee set for this specific order.
