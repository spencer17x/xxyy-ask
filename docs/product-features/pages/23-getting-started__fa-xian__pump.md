# Pump

Pump列表会实时刷新最新的Pump项目，并按照代币创建时间进行排序，会同时展示项目的市值、创建距离、时间以及 Pump 进度 。点击任意项目，会跳转项目的K线以及详细信息，方便查看该代币是否有交易的机会

Pump列表支持按时间排序以及按进度排序，按进度排序会默认将进度最大的优先展示。方便查看Pump即将打满的项目
