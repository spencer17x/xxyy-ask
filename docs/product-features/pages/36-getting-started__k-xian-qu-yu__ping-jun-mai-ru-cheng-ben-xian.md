# 平均买入成本线

<figure><img src="/assets/xxyy-docs-reGhTxjhXRct4gsxivFL.png" alt=""><figcaption></figcaption></figure>

XXYY 会根据每一笔的买入成本计算平均成本线，如果有多次买入就会计算多次买入后的平均成本，方便衡量自身持仓成本。
