# TagHolder

XXYY提供 Tag Holder查询，可直接查看关注地址在该代币的买入情况以及持仓情况，方便第一时间判断该项目是否值得买入或卖出。

如Tag Holder中，持仓量小于1，则说明该地址已经清仓，代表其曾历史买入过该项目
