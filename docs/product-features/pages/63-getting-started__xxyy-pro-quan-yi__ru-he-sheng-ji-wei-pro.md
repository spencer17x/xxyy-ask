# 如何升级为 Pro

点击右上角个人中心的会员积分，可查看当前地址积分数量。

交易积分是根据当前地址下所有交易地址的买入卖出笔数、买入卖出金额综合计算。

积分会在每天UTC  0点准时空投到地址账户内
