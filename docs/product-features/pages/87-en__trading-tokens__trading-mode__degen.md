# Degen

Degen Mode offers the absolute fastest on-chain speed possible! We're talking millisecond-level execution, ensuring you're always one step ahead.

However, this raw speed comes with a trade-off: Degen Mode does not guarantee sandwich attack protection. It is recommended to use this mode with a low slippage tolerance to mitigate risk.

Note: The minimum transaction fee (Gas) for Degen Mode is 0.002 SOL.
