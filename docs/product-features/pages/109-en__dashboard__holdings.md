# Holdings

The Holdings section provides a comprehensive overview of every asset you have purchased with your currently selected wallet. By default, the list is sorted in descending order by the current value of your holdings, placing your most valuable assets at the top for easy viewing.

Managing Visibility

* Hiding a Token: If you wish to hide a specific token from this main view, you can click the visibility icon on the far right of its row. Click the icon again to unhide it.
* Viewing All Tokens: To see a complete list of every token, including those you have manually hidden, click the "Show All" button.

Filtering Small Balances

At the top of the page, you will find the "Hide small Tokens" checkbox. This feature helps you de-clutter your view by hiding assets with a very low current value.

* You can customize the threshold value that defines a "small token." The default setting is ≤0.001.
* This threshold is based on the native currency of the currently viewed blockchain (e.g., 0.001 SOL on Solana, 0.001 ETH on Ethereum).
* When this box is checked, any token with a current value below your set threshold will be hidden.
* Important: Unchecking this box will reveal your complete purchase history, including tokens you have already sold (i.e., those with a zero balance).
