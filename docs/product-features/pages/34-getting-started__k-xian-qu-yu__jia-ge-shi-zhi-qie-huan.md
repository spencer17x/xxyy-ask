# 价格市值切换

K 线单位支持展示价SOL价格或USD价格，并且支持市值和代币价格的切换，通常看代币价格会采用 SOL价格，看市值会采用USD价格。如发现 K 线价格和右侧代币信息区价格不一致，可点击切换按钮保持一致。
