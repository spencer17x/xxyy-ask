# Limit Orders

With XXYY's Limit Order feature, you can set your trades in advance to automatically buy or sell tokens when they reach your target price.

Order Conditions\
Set up your trade by choosing one of the following trigger conditions:

* Price Rises By (%): Place an order when the current price increases by a specified percentage.
* Price Drops By (%): Place an order when the current price decreases by a specified percentage.
* Market Cap Rises By (%): Place an order when the token's market cap increases by a specified percentage.
* Market Cap Drops By (%): Place an order when the token's market cap decreases by a specified percentage.
* Price Reaches: Place an order when the token's price hits a specific target value.

Order Duration (Hours)\
Set the duration for which your limit order will remain active. If the conditions are not met within this time frame, the order will automatically expire.
