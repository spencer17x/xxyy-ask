# On-Chart Transaction Markers

The chart can display visual markers for key trading activities. You can toggle between two primary modes: viewing your own transactions or tracking the Dev's on-chain activity.

Furthermore, these markers can be used in conjunction with the "Latest Trades" and "Top Holders" lists. Simply select a wallet from those lists to see all of its buy and sell points plotted directly on the chart.

The following markers are used:

* Buy: Marks one of your buy transactions.
* Sell: Marks one of your sell transactions.
* Dev Buy: Marks a buy transaction from the Dev's wallet.
* Dev Sell: Marks a sell transaction from the Dev's wallet.
* Raydium Launch: A special event marker indicating the exact moment a project's liquidity was successfully added to Raydium.
