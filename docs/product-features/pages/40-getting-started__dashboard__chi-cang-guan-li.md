# 持仓管理

持仓管理会展示当前交易地址购买的所有交易项目，并按照持有中的 Sol 价值倒序进行排列。方便查看当前交易地址中持仓量最高的代币项目

持仓管理支持隐藏项目，点击项目最后最右侧显示按钮即可隐藏或取消隐藏。

在持仓管理顶部可选择是否隐藏小额代币，并且可设置对于小额的定义，默认是0.001。如取消勾选，还可查看之前购买过的项目。

点击展示所有代币按钮，会显示包括隐藏和未隐藏的所有项目
