# 快捷交易

<figure><img src="/assets/xxyy-docs-s1GC6881bft4g0hiy57M.png" alt=""><figcaption></figcaption></figure>

XXYY支持快捷交易面板设置，方便一键快速购买或卖出项目。点击右侧的悬浮按钮，可将快捷交易面板悬浮在页面上自由移动

快捷交易面板按钮设置，可点击面板右上角的设置按钮，自定义设置每个按钮的买入、卖出金额以及交易滑点交易费。右侧交易面板勾选的交易模式，会直接影响快捷交易的模式选择和滑点交易费设置

* 默认买入金额（Default Buy Amount）：0.1 SOL，0.2 SOL，0.5 SOL，1 SOL，四个默认值均支持修改，设置完成后交易组件中挂单买入默认使用该值
* 默认卖出比例（Default Sell Ratio）：10%，25%，50%，100%，四个默认值均支持修改，设置完成后交易组件中挂单卖出默认使用该值
