# Graduated Sell

For tokens you have acquired on pump.fun, the Graduated Sell feature allows you to create an automated sell order before the token's liquidity is added to Raydium. You can customize the exact percentage of your holdings you wish to sell.

Once the order is successfully created, our system will monitor the blockchain. The instant it detects that the token has "graduated" and its liquidity pool has been added to Raydium, your sell order will be executed immediately.
