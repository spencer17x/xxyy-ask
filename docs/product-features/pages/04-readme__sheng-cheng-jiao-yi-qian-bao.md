# 生成交易钱包

为了保证交易安全和便捷，XXYY 使用新生成的交易钱包进行交易。首次登录XXYY，你需要生成至少一个交易钱包。

* 生成交易钱包：在导航上选择要交易的链，点击生成按钮，生成交易钱包。
* 保存生成的钱包和私钥：钱包生成后，你将可以看到钱包地址和私钥。私钥会在你主动点击“查看私钥”后才会展示。每个钱包的私钥仅会展示一次，请务必保存好你的私钥，私钥丢失后无法找回。
* 转入资产：钱包生成后，你需要在钱包中转入资产后才能交易，你可以使用其他钱包扫码转入或者复制地址转入资产，资产转入完成后即可开始交易。
