# 搜索

<figure><img src="/assets/xxyy-docs-PihNl8dueMsgByFnDrqF.png" alt="" width="375"><figcaption></figcaption></figure>

XXYY 提供了强大的搜索功能，支持输入代币合约地址，交易对地址，名称搜索代币。其中代币名称支持模糊搜索，能够帮助你快速找到目标代币。
