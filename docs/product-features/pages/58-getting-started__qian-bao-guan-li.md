# 钱包管理

<figure><img src="/assets/xxyy-docs-8nkvbFcMNEEQfWA0Tw2i.png" alt=""><figcaption></figcaption></figure>

XXYY 支持每个用户每条链最多创建100个交易钱包。Pro 用户最多创建500个交易钱包。创建钱包后，可在钱包管理列表中查看已经创建的钱包。支持将已经创建的钱包置顶，交易时将优先展示置顶的钱包。

#### 资金转账

资金转入：支持将链接钱包的资金快速转账至交易钱包，提交转账后可以在转账历史中查看转账状态。

资金转出：支持将交易钱包的资金快速转账至链接钱包，提交转账后可以在转账历史中查看转账状态。
