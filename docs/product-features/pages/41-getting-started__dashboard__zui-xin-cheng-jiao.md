# 最新成交

<figure><img src="/assets/xxyy-docs-R0XknhDckz2koDKHhPym.png" alt=""><figcaption></figcaption></figure>

最新成交展示了当前项目所在池子的每一条交易记录，平均展示速度和链上差距1秒以内，方便查看最新的成交记录。最新成交支持查看该项目所有成交记录。

点击每一行成交记录后面的筛选按钮，即可筛选此地址在当前项目所在池子的所有成交记录。

点击每一行成交记录后面的浏览器按钮，即可查看当前交易在 Solscan 的 Hash值。

点击每一行成交记录后面的关注按钮，即可关注当前地址设置、备注名以及推送规则，具体可跳转监控管理查看详细设置教程。

最新成交顶部筛选，支持交易时间倒序，支持筛选交易类型，支持切换 SOL价格或 USD 价格，支持筛选交易大小额以及成交价格，支持筛选指定交易地址的所有成交记录
