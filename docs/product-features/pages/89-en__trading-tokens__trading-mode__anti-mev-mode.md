# Anti-MEV Mode

Anti-MEV Mode offers the most robust protection available, ensuring your transactions are shielded from sandwich attacks and other forms of MEV exploitation for a secure on-chain experience.

This mode is ideal for high-stakes scenarios where security is paramount, such as launch snipes or executing a "Follow Dev Sell." Because it provides this high level of security, you can safely use it with a high slippage tolerance (e.g., 2000%) to ensure your transaction succeeds without being exploited.
