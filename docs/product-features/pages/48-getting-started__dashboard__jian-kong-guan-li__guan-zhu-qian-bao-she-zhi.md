# 关注钱包设置

钱包地址：输入你要关注的钱包地址

钱包名称：设置一个你对该钱包的备注或昵称，如DNF

钱包分组：可手动填写钱包分组或选择钱包分组，通过分组可以实现批量管理

#### &#x20;**钱包交易推送设置：**

钱包交易类型：设置推送买入或卖出

最小买入金额：设置最小买入金额，如低于该金额则不推送

最小卖出金额：设置最小卖出金额，如低于该金额则不推送

是否开启推送：对于一些只关注不想看推送的钱包，可设置关闭推送

仅推送Pump交易：可设置是否仅推送Pump内盘交易
