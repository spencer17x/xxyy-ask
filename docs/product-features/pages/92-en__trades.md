# Trades

The Trades section is your primary hub for discovering and analyzing trading opportunities. For a flexible and efficient user experience, the interface is organized into two main parts: a top navigation bar and a main content view below.

The top bar gives you access to five powerful discovery modules:

* Watchlist
* New Pairs
* Trending
* Monitor
* X Tracker

The area below the navigation bar displays the detailed content for the module you have selected. For a more compact layout, you can click the collapse icon on the right-hand side to hide the main content view, leaving only the top navigation modules visible.
