# Holder

The Holders tab provides a comprehensive list of all addresses that currently hold the token, ranked by the size of their position. This allows you to analyze token distribution and identify key players. You can also filter this list to see Tag Holders or KOL wallets.

The main table includes the following columns:

* Rank: The real-time rank of the wallet based on its holding size.
* Wallet: Displays the wallet address.
  * If you are monitoring a wallet, its custom name will be highlighted here.
  * It also shows the total number of buy (in) and sell (out) transactions.
  * Special tags (e.g., Insider, New, Sniper, Dev) are displayed to help you quickly identify wallets with specific behaviors.
* Amount: The total number of tokens held by this address.
* Value: The current market value of the holdings in USD.
* %(Real): This shows the percentage of the total token supply held by the address. The (Real) value is a special XXYY calculation that tracks and aggregates holdings across wallets that appear to be linked through batch transfers. This helps reveal the true concentrated ownership of a single entity, which will be highlighted.
* SOL Source: Identifies the source of the wallet's first SOL deposit, which can indicate if it was funded by a Centralized Exchange (CEX) or another wallet.
* Token Source: Shows how the wallet first acquired this token, either through a "Buy" transaction or a "Transfer" from another address.
* Balance (SOL): Displays the wallet's total current SOL balance. This can help you identify potential whale wallets or those with significant capital.
* TF time (Last Transaction Time): Indicates the last time this wallet performed a transaction (buy, sell, or transfer) with this specific token.
