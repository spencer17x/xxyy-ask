# Swap 交易

XXYY 支持一键买卖代币，操作简单，毫秒级上链。

* 选择钱包：选择你想交易的钱包，可以看到对应钱包余额。
* 交易金额：可以自定义买入的SOL数量或者卖出代币的比例，点击买入或卖出按钮后将立即发起交易。
* 交易通知：交易发起后，你可以看到交易进行中和交易完成的通知弹窗。
* 交易设置：包括滑点、交易 Fee(Gas+Tip)，支持自定义，也可以选择自动设置，会根据链上情况动态调整，保障交易速度。
