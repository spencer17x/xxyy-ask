# 趋势

<figure><img src="/assets/xxyy-docs-6Lsf5Km990aod15t0wpT.png" alt="" width="306"><figcaption></figcaption></figure>

趋势列表是XXYY的一大亮点功能。

目前有1分钟、5分钟、1小时、6小时、24小时五种趋势

支持查看仅 Pump 项目

趋势的排序是根据XXYY的智能算法进行计算，优先展示趋势热度最高的项目，同时支持按照24小时的价格涨跌幅进行排序，或者仅查看Pump的项目，所有的Pump项目会在项目名称后面有 Pump 的 Logo标识
