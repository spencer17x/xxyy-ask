# 开盘狙击

搜索代币合约，如果代币尚未添加Raydium流动性池，或者Pump项目Bonding Curve已经打满，等待流动性迁移时，可以设置狙击订单。当代币添加Raydium 池子后，会第一时间发起狙击买入！

输入买入金额，点击确认，即可创建狙击任务。
