好的筛选会让扫链事半功倍！

社交媒体：查看当前项目是否包含Twitter、Telegram、Website等信息

创建时间（分钟）：设置查看代币创建时间

市值：设置指定市值范围内的 Pump 项目

Dev Buy金额：根据 Dev 买入金额过滤筛选项目

Dev Sell金额：根据 Dev 卖出金额过滤筛选项目

Holder人数：根据 Holder 人数范围 过滤筛选项目
