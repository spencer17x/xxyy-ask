# Trading Settings

You can access the Trading Settings page by clicking "Settings" in the top-right menu. Here, you can configure the following parameters, which will take effect immediately once set.

* Default Slippage Tolerance\
  You can set a custom default slippage percentage. This value will be automatically used in the trade interface for all your transactions.
* Default Trading Mode\
  You can select your preferred default trading mode (e.g., Degen, Turbo, or Anti-MEV). The mode you choose here will be the pre-selected option in the trade interface.
* Default Priority Fee\
  (Unit: SOL)\
  This fee, which includes the base gas fee and an additional tip, is paid to the network to prioritize your transaction. You can set a custom default Priority Fee here, and this amount will be used as the default for all your trades.
