# Swap

XXYY makes swapping tokens simple and fast. Execute trades instantly with a single click and benefit from near-instant on-chain confirmation.

1\. Select Your Wallet\
Choose the wallet you wish to trade from. The current token balance for the selected wallet will be displayed.

2\. Enter the Trade Amount\
You can either specify the amount of SOL you want to buy or select a percentage (e.g., 25%, 50%, 100%) of the token you want to sell. Clicking the "Swap" button will immediately execute the trade.

3\. Transaction Notifications\
Once the transaction is submitted, you will receive real-time pop-up notifications to keep you updated, showing when your transaction is "Processing" and when it is "Completed."

4\. Transaction Settings\
For more control, you can adjust your transaction settings. This includes:

* Slippage Tolerance: Set the maximum percentage of price change you are willing to accept.
* Transaction Fee (Gas + Tip): This fee influences how quickly your transaction is processed on the blockchain.

You can set these values manually or select the "Auto" mode. The "Auto" setting dynamically adjusts the parameters based on current network conditions to help ensure your transaction is fast and successful.
